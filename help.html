Help for Very Bad Game

We understand the game can be confusing at times. Here's some help.

Frequently Asked Questions:

How do I play?

Use the keyboard and mouse to control your character. The game is intentionally difficult and strange, so expect the unexpected.

The game isn't running. What should I do?

Make sure your system meets the minimum requirements. If you're having issues, try restarting the game or updating your graphics drivers.

Where can I get help with bugs?

If you're experiencing technical issues, head to our Discord server (link) or check our FAQ page.
